# 视频处理技术学习网站 - 知识可视化重构

一个采用流程图、交互图、时序图、树图、表格等可视化方式表达视频处理知识的静态网站。

## 重构目标

将视频处理知识从纯文字表达转换为**可视化图表表达**，保持静态网站架构，提升知识理解效率。

## 重构方法

采用以下可视化工具重构知识表达：

### 1. 流程图 (Flowchart)
- 展示编码流程、数据流向
- 说明算法步骤、处理过程
- 示例：DCT变换编码流程、运动估计流程

### 2. 树图 (Tree Diagram)
- 展示分类、层级关系
- 说明技术特性、知识结构
- 示例：色彩空间分类、编码标准对比

### 3. 时序图 (Sequence Diagram)
- 展示时序关系、交互流程
- 说明GOP结构、数据传输
- 示例：GOP帧类型序列

### 4. 对比表格 (Comparison Table)
- 对比不同技术特性
- 说明参数、配置差异
- 示例：编码标准对比、质量评估指标

### 5. 知识卡片 (Knowledge Card)
- 总结核心要点
- 展示公式、关键概念
- 示例：转换公式、优化目标

## 技术实现

- **Mermaid图表库**：通过CDN引入，支持流程图、树图、时序图等
- **自定义CSS样式**：优化图表展示效果
- **静态HTML架构**：保持简单、快速的加载特性
